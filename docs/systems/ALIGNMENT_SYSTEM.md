# Alignment System

## Dual-Axis Morality

Two independent axes track moral choices:

```
                    COMPASSION
                        ↑
                        |
    Harsh ←─────────────┼────────────→ Merciful
    (-100)              │              (+100)
                        ↓
                      CRUELTY
```

```
                      HONOR
                        ↑
                        |
   Deceitful ←──────────┼────────────→ Honorable
    (-100)              │              (+100)
                        ↓
                     PRAGMATIC
```

## Honor Axis (How You Act)

| Level | Name | Description |
|-------|------|-------------|
| +100 | **Paragon** | Never lies, always keeps promises |
| +50 to +99 | **Honorable** | Generally honest and fair |
| -49 to +49 | **Pragmatic** | Does what's necessary |
| -50 to -99 | **Deceitful** | Regularly manipulates |
| -100 | **Serpent** | Master manipulator |

## Compassion Axis (How You Treat Others)

| Level | Name | Description |
|-------|------|-------------|
| +100 | **Saint** | Never harms innocents, always merciful |
| +50 to +99 | **Merciful** | Generally kind and forgiving |
| -49 to +49 | **Balanced** | Judges each situation |
| -50 to -99 | **Harsh** | Punishes severely |
| -100 | **Tyrant** | Cruelty is a tool |

## Purity Rewards

### Incremental Buffs

| Threshold | Buff |
|-----------|------|
| **\|25\|** | +5% alignment-related ability effectiveness |
| **\|50\|** | Unique dialogue options unlock |
| **\|75\|** | +10% effectiveness, minor passive ability |
| **\|90\|** | +15% effectiveness, alignment ability unlocks |
| **\|100\| PURE** | Ultimate alignment class unlocks |

### Purity Lock
One choice against your axis resets progress toward that extreme. At 100%, a single betrayal permanently locks you out of that alignment's ultimate class.

## Companion Loyalty Modifiers

Alignment affects companion loyalty:
- **Matching alignment:** +loyalty
- **Conflicting alignment:** -loyalty
- **Extreme actions:** Can trigger breaking points
